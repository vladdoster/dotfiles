# Mac OS (darwin)

- Install CLI/GUI programs via HomeBrew
- Systemwide config via Applescript

> [!NOTE] sudo access is required or errors will occur.

## Usage

#### Install dotfiles

```bash
make dotfiles
```

#### Setup a new system

```bash
make setup
```

#### List possible targets

```bash
make list
```

#### Change shell

```bash
sudo usermod --shell /bin/zsh
```
